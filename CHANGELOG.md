# Changelog

All notable changes to JargoNaut will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

### Added
- Comprehensive README.md with installation instructions and screenshots

## [0.4.0] - 2024-12-01

### Changed
- Updated Chrome extension to v0.4.0
- Added justification for activeTab permission
- Improved AI prompt for better jargon detection

### Fixed
- Circular button icon (removed white corners)

## Release Notes

### Version Numbering
- **Chrome:** Uses semantic versioning (0.4.0)
- **Safari:** Uses Apple's versioning format (1.0.0(2))
- Both versions are functionally equivalent

### Platform Support
- Chrome: Windows, macOS, Linux
- Safari: macOS 11.0+

### API Requirements
- OpenRouter API key (free tier available)
- Internet connection for AI processing
